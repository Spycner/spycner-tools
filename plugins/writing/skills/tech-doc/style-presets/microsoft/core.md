# Microsoft: Core Style

Source: https://learn.microsoft.com/en-us/style-guide/welcome/
License: Proprietary; paraphrased for fair-use reference
Last refreshed: 2026-04-29

## Voice and tone

- Warm and relaxed, crisp and clear, ready to lend a hand.
- Project friendliness without being frivolous.
- Write like you speak. Read text aloud. Avoid jargon and overly complex language.

## Person

- Use second person ("you").
- Use active voice. Make the actor explicit.

## Voice and tense

- Use present tense for current capabilities and behaviors.
- Avoid future-tense scaffolding ("will be able to", "is going to").

## Sentence structure

- Lead with the most important. Front-load keywords for scanning.
- Bigger ideas, fewer words. Prune every excess word.
- Start statements with verbs. Edit out "you can" and "there is/are/were".

## Contractions

- Use contractions (it's, you'll, you're, we're, let's) to project friendliness.

## Headings and capitalization

- Sentence case: capitalize only the first word and proper nouns.
- Never use title case.
- Skip end punctuation on titles, headings, and subheadings of three or fewer words.
- Front-load keywords.

## Punctuation

- Use the Oxford comma in lists of three or more items.
- One space after periods, question marks, and colons.

## Future features

- Avoid future-tense scaffolding ("will be able to", "is going to").
- Get to the point fast; don't announce what you are about to say.

## Global audience

- Write for an international audience.
- Avoid idioms, regional expressions, sports metaphors, and cultural references.
- Use simple constructions and short sentences.
